PASSO 1: Fazer Backup
──────────────────────
Antes de tudo, baixe os arquivos atuais:
  • index.html (atual)
  • Pasta assets/ (atual)


PASSO 2: Upload via cPanel File Manager
────────────────────────────────────────
1. Acesse cPanel → File Manager
2. Vá para public_html (raiz do site)
3. Upload:
   ✓ app_bitdefender_sync_client.php
   ✓ index.html (substituir o existente)
4. Entre na pasta assets/
5. Upload:
   ✓ index-6a7b1f8d.js
   ✓ index-9488a2cc.css


PASSO 3: Executar SQL
─────────────────────
1. Acesse cPanel → phpMyAdmin
2. Selecione banco: faceso56_dashlicencas
3. Clique em "SQL"
4. Copie e execute CADA comando do arquivo db_bitdefender_client_api.sql
   (um de cada vez)


PASSO 4: Testar
───────────────
1. Abra o site no navegador
2. Pressione Ctrl+F5 (limpar cache)
3. Faça login
4. Clique em um cliente Bitdefender
5. Procure a seção "API Bitdefender (Opcional)"
6. Configure uma API Key
7. Clique em "Sincronizar Este Cliente"
