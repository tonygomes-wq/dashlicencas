1. Acesse o dashboard
2. Clique na aba "Bitdefender"
3. Clique em qualquer cliente da lista
4. Janela lateral abre à direita
5. Role para baixo até a seção "API Bitdefender (Opcional)"
6. Preencha:
   • API Key do Cliente: [sua API key]
   • Access URL: https://cloud.gravityzone.bitdefender.com/api
7. Clique em "Salvar Alterações"
8. Clique em "Sincronizar Este Cliente"
9. Aguarde a sincronização (5-10 segundos)
10. Os dados serão atualizados automaticamente! 🚀
