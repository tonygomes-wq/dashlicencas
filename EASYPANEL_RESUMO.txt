🚀 INÍCIO RÁPIDO (3 PASSOS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASSO 1: Preparar Arquivos
───────────────────────────

No Windows (Git Bash):
  bash preparar_easypanel.sh

Ou manualmente:
  1. Criar pasta: db_init/
  2. Exportar banco do phpMyAdmin
  3. Salvar como: db_init/init.sql


PASSO 2: Criar Repositório Git (Opcional)
──────────────────────────────────────────

git init
git add .
git commit -m "Initial commit - Dashboard Licenças"
git remote add origin https://github.com/seu-usuario/dashlicencas.git
git push -u origin main


PASSO 3: Deploy no Easypanel
─────────────────────────────

1. Login no Easypanel
2. Create Service → App → From Git Repository
3. Cole URL do repositório
4. Configure variáveis de ambiente:
   
   DB_HOST=db
   DB_NAME=dashlicencas
   DB_USER=dashlicencas_user
   DB_PASSWORD=SuaSenhaSegura123!

5. Adicione domínio
6. Deploy!

⚡ DEPLOY LOCAL (Alternativa)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Se você tem Docker instalado localmente:

1. Edite docker-compose.yml (altere senhas)
2. Execute:
   
   docker-compose up -d

3. Acesse: http://localhost
