🔍 DIAGNÓSTICO AVANÇADO (Se nenhuma versão funcionar)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Se NENHUMA das 3 versões funcionar, o problema é outro:

1. Verifique a estrutura de pastas no cPanel:
   
   Deve ser EXATAMENTE assim:
   
   public_html/
   └── dashlicencas/          ← Ou pode estar direto em public_html/
       ├── index.html
       ├── assets/
       │   ├── index-f981d49f.js
       │   └── index-bda104fb.css
       └── app_bitdefender_sync_client.php

2. Teste a URL direta no navegador:
   
   Tente acessar:
   https://macip.com.br/dashlicencas/assets/index-f981d49f.js
   
   Ou:
   https://macip.com.br/assets/index-f981d49f.js
   
   Se aparecer código JavaScript = Arquivo existe!
   Se aparecer erro 404 = Arquivo NÃO existe ou está em lugar errado!

3. Verifique o tamanho dos arquivos:
   
   No File Manager, clique nos arquivos e veja o tamanho:
   - index-f981d49f.js deve ter ~1.010.000 bytes (987 KB)
   - index-bda104fb.css deve ter ~57.000 bytes (56 KB)
   
   Se estiverem muito menores, o upload falhou!

4. Verifique as permissões:
   
   Clique com botão direito → Permissions
   Deve ser: 644 (rw-r--r--)
